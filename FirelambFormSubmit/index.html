Instroduce 插件介绍

jQuery Form Submit is a practical jquery plugin.You can use ajax post method or direct post when you click the button to submit the form and
at the same time you can encrypt the selector of the form.

jQuery Form Submit 是一款实用的jquery插件，你可以在按钮点击提交表单的时候选择ajax提交或直接提交，并且你可以选择加密表单中的选择器

Structue文档结构

assets,dist------just some css,js files of this demo,not core.只是些用于此插件演示的css,js文件，不是核心文件

FirelambFormSubmit-----core files 插件的核心文件

------------------core: core files核心文件

------------------lib: the third plugins 第三方插件

------------------demo: the demo files of the ajax post method （ajax方式提交时需要的演示文件）

------------------ddashboard.css,navbar-fixed-top.css: just the css files of index.html 只是index.html需要的样式文件

jquery-1.10.1.min.js------of course you won't forget 你不会忘记的jquery文件

Options 插件配置

Option Description
formId the Id of the form that you wanna post
你需要提交的form的id
isEncrypt Encrypt or not,just use true or false
指定加密或者不加密，true,false
encryptObjs the array of the selectors's id that need be encrypted in the form
表单中需要加密的选择器的id，以数组形式
encryptType the encrypt types,include MD5，SHA1，SHA256，SHA3， AES,DES,Rabbit,RC4
加密方法，包括MD5，SHA1，SHA256，SHA3， AES,DES,Rabbit,RC4
submitType the submit type,you can use 'ajaxPost' or 'dirPost'
表单提交的方式 可以使用'ajaxPost' or 'dirPost'
postUrl the target url when you use 'ajaxPost' that the form post to
当使用'ajaxPost'时 表单提交的目标地址
ajaxTips the tips text when you use 'ajaxPost' and the form is sending
当使用'ajaxPost'时 表单正在提交是给出的提示文件
beforePost call a function before post the form,such as validate the format.The important thing is that
you should remember return a 'true' or 'false',so that the plugin can judge post the form or not.
表单提交前执行的函数，例如可以进行格式验证等 记得返回true或者false 以便插件判断是否提交表单
callEvent callback when you use 'ajaxPost' ,in the success function
当使用'ajaxPost'时 success函数中调用的回调函数
